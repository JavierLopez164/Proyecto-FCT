# Proyecto FCT 
Este proyecto tiene como objetivo modernizar las cartas tradicionales de un restaurante mediante la implementación de códigos QR que permitan a los clientes acceder al menú digital desde sus dispositivos móviles.
La solución busca mejorar la experiencia del cliente, optimizar la gestión del menú y reducir costos operativos asociados a la impresión física.
##  Tecnologías Utilizadas
Tecnologías utilizadas en el proyecto:
###  Frontend
- **Angular** – Framework para el desarrollo de la aplicación web.
- **Tailwind CSS** – Framework de estilos para diseño moderno y responsive.
- **Typescript** – Framework de estilos para diseño moderno y responsive.
### Aplicación Móvil
- **Kotlin Multiplatform** – Desarrollo multiplataforma compartiendo lógica entre Android y otros entornos.

### Backend
- **Java Spring Boot** – Framework para la creación de APIs REST seguras y escalables.Con la implementación de Hibernate ,Spring Security

###  DevOps & Contenedores
- **Docker** – Contenerización de la aplicación para facilitar despliegue y portabilidad.
